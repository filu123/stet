# feature: editor

The TipTap editor: editor component, toolbar/bubble menu, custom extensions.

- `components/` — editor UI (all `"use client"`, `immediatelyRender: false`)
- `hooks/` — editor logic (e.g. `useDocumentEditor`, `useAutosaveDocument`)
- `extensions/` — TipTap/ProseMirror extensions (highlight pills, callouts, AI decorations)
